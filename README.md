# 📝 My Task Manager

A simple, beginner-friendly Task Manager web application built with HTML, CSS, and JavaScript.

---

## 🚀 How to Use

1. **Create a project folder** on your computer (e.g., `task-manager/`).

2. **Create three files** inside the folder:
   - `index.html`
   - `style.css`
   - `script.js`

3. **Copy and paste** the corresponding code into each file.

4. **Open `index.html`** in your web browser by:
   - Double-clicking the file, **or**
   - Right-clicking → "Open with" → Choose your browser, **or**
   - Dragging the file into a browser window.

5. **Start adding tasks** and explore all the features!

---

## ✨ Features Included

| Feature              | Description                                          |
| -------------------- | ---------------------------------------------------- |
| ✅ **Add Tasks**      | Type a task and click "Add" or press Enter           |
| 🎨 **Priority Levels**| Low (green), Medium (yellow), High (red)            |
| ☑️ **Mark Complete**  | Click the checkbox to toggle task completion         |
| 🗑️ **Delete Tasks**  | Click the ✕ button to remove a task                 |
| 🔍 **Filter Tasks**  | View All, Active, or Completed tasks                 |
| 📊 **Live Stats**    | See total, active, and completed task counts         |
| 💾 **Local Storage** | Tasks persist even after closing the browser         |
| 📱 **Responsive**    | Works on mobile, tablet, and desktop devices         |
| 🎭 **Animations**    | Smooth slide-in animations for newly added tasks     |
